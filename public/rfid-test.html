üè∑Ô∏è Jenis Tag RFID yang Kompatibel

ISO14443 Type A

Contoh: MIFARE Classic, MIFARE Ultralight, MIFARE DESFire

Fitur: High-frequency RFID, jarak baca 1-10 cm

Penggunaan: Transport cards, access control, payment cards

ISO14443 Type B

Contoh: Calypso, DESFire

Fitur: Secure communication, encryption support

Penggunaan: Government ID cards, secure access

ISO15693

Contoh: I-CODE, Tag-it

Fitur: Longer read range (up to 1 meter)

Penggunaan: Inventory tracking, asset management

FeliCa

Contoh: Sony FeliCa

Fitur: High-speed communication

Penggunaan: Japanese transport cards, mobile payments

üîß Cara Testing RFID

Langkah 1: Persiapan

  • Pastikan menggunakan Chrome di Android
  • Aktifkan NFC di pengaturan perangkat
  • Buka aplikasi MarketPlay

Langkah 2: Akses Scanner

  • Buka halaman Scanner (ikon NFC di bottom nav)
  • Pilih mode "RFID"
  • Pastikan status "RFID Scanner" aktif

Langkah 3: Scan Tag

  • Klik tombol "Scan RFID Tag"
  • Dekatkan tag RFID ke bagian belakang perangkat
  • Jaga jarak 1-10 cm
  • Tunggu hingga tag terdeteksi

Langkah 4: Lihat Hasil

  • Tag ID akan ditampilkan
  • Klik "Detail Tag" untuk informasi lengkap
  • Riwayat scan tersimpan otomatis

‚ö†Ô∏è Troubleshooting

Tag tidak terdeteksi

  • Pastikan NFC aktif di pengaturan
  • Coba dekatkan tag lebih dekat (1-5 cm)
  • Pastikan tag tidak rusak atau terkelupas
  • Coba tag RFID lain untuk memastikan

Error "Browser tidak mendukung"

  • Gunakan Chrome di Android
  • Pastikan versi Chrome 89 atau lebih baru
  • Coba restart browser
  • Pastikan menggunakan HTTPS

Tag ID tidak muncul

  • Beberapa tag mungkin tidak memiliki serial number
  • Coba tag RFID yang berbeda
  • Pastikan tag dalam kondisi baik
  • Cek apakah tag memiliki data NDEF
